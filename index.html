Despertador Matemático

Sistema embebido diseñado para garantizar que el usuario despierte de manera efectiva mediante la resolución de operaciones matemáticas. El dispositivo permite configurar hasta 7 alarmas programables a través de una interfaz web. Incluye un servomotor que actúa como cerradura eléctrica para desbloquear el botón de apagado solo cuando la respuesta es correcta. La operación cambia automáticamente cada 60 segundos si no se ha resuelto, y el sistema cuenta con señales acústicas y visuales para alertar al usuario. Utiliza conexión WiFi para sincronizar la hora mediante NTP y dispone de batería de respaldo para cortes de energía.

Dispensador de comida para aves

Sistema embebido con conectividad IoT diseñado para dispensar comida de manera programada, proporcionando alimento en intervalos regulares. El dispositivo utiliza un ESP32 como microcontrolador principal, conectado a una red WiFi, y se integra con la plataforma Ubidots para monitoreo y control remoto desde cualquier lugar. La hora se sincroniza mediante NTP, permitiendo programar horarios específicos de alimentación. Cuando el contador de porciones llega a cero, se envía una alerta a Ubidots para notificar que el contenedor debe ser rellenado.
