Explore some of my proyects

Here you will find projects that i've been working on recently. You can find more info in my GitHub profile.

Math Alarm Clock with ESP32

I designed and built an interactive math-based alarm clock using an ESP32, which forces users to solve random arithmetic problems to disable the alarm. The system combines embedded programming, 3D printing, and IoT connectivity for a unique wake-up experience.

Automated Bird Feeder with IoT

Developed using an ESP32, dispenses food at scheduled times and monitors food inventory via the IoT platform Ubidots. 3D printed body. The system includes: Remote control via a mobile app. OLED display to show time and feeder status. Audible notifications (buzzer) during dispensing. Manual container reset via a physical button.

Repair and Modernization of an iHR550 Oscilloscope

Successfully restored a malfunctioning iHR550 oscilloscope for my university by designing a custom PCB and developing an Arduino-based control system for its internal mirror assembly. This project combined hardware repair, reverse engineering, and embedded programming to bring legacy equipment back to life.
